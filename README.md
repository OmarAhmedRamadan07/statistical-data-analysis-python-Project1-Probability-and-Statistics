# Statistical Data Analysis & Frequency Distribution 📊

A Python-based data analysis project designed to organize, summarize, and visualize a dataset of 100 student achievement test scores applying for a master's degree.pdf].

Developed as an academic mini-project for **GEN211 - Probability and Statistics (Fall 2025)** at **El Sewedy University of Technology (SUT)**.pdf].

---

## 🚀 Project Objectives & Features

This project demonstrates core competencies in descriptive statistics and data visualization using Python:

### 1. Data Organization & Summarization
* **Frequency Distribution Table:** Dynamically constructs an 8-class frequency table including class limits, class boundaries, frequencies, relative frequencies (%), cumulative frequencies, and cumulative relative frequencies.pdf, report project 1 probablty.docx].
* **Raw Data Statistics:** Calculates the exact Mean, Standard Deviation (SD), and Coefficient of Variation (CV) for the raw dataset of 100 scores.
* **Grouped Data Statistics:** Computes the approximated Mean, Standard Deviation, and CV using the grouped frequency distribution midpoints and compares them with the raw data.

### 2. Data Visualization
Utilizes Python plotting libraries to visually represent the data distribution:
* **Histogram:** To show the frequency of scores within each class interval.pdf].
* **Frequency Polygon:** To highlight the shape and trend of the data distribution.pdf].
* **Ogive Plot:** A cumulative frequency graph to easily identify percentiles and data accumulation.pdf].
